## Run Locally

Clone the project

```bash
  git clone hhttps://github.com/manojsagar066/voice_billing.git
```

Go to the project directory

```bash
  cd server
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
